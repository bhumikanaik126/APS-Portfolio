## üí° Algorithmic Insight

### Goal:

Use **Dijkstra‚Äôs Algorithm** to find minimum-cost paths from source nodes (user location, entry page, content node) to all other nodes.

### Core Steps:

1. Initialize distances from source to all nodes as ‚àû except source (0).
2. Use a min-priority queue to repeatedly select the node with the smallest distance.
3. Relax edges: update distances of neighbors if a shorter path is found.
4. Repeat until all nodes are processed.
